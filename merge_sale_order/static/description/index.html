Merge Sale Orders


Merge two or Many sale orders

The purpose of this module is to merge sale order merge with are in Request For Quoation State.

Open the Merge sale Order

By clicking the merge sales orders menu you can see this wizard for merge the sale orders.



  • First option 'Create new order and cancel all selected sale orders' will create and other selected order will be cancelled.

  • Second option will create and other selected order will be deleted.

  • If we goes with third option.You can see a 'Sale Orders' field, In this you can select a existing order in which you want to added another selected orders.After click on Merge Orders button, You can see a another orders merge with selected existing order.And rest of all selected orders will be cancelled.

Merge Sale orders

Merge all selected sale order which has same partner and all the order must in draft state otherwise it will raise warning.
